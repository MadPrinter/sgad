# Spec: SentinelOps Control Plane

## Requirements

- First viewport shows product identity and live operational state.
- KPI strip includes Active Agents, Blocked Tasks, Gate Failures, and Confidence.
- Agent Activity table includes status, task, risk, and evidence columns.
- Governance Gates panel includes spec, design review, evidence matrix, and rollout approval.
- Rollout Evidence timeline is visible.
- Page is responsive and includes accessibility basics.
