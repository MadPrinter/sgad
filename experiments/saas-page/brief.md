# Brief: AgentOps Command Center

Build a single-page SaaS dashboard for an AI operations team.

Required page capabilities:

- first viewport must show the product and live operational state
- left navigation or clear app shell
- KPI strip for active agents, blocked tasks, gate failures, and rollout confidence
- agent activity table
- governance gate status
- rollout or evidence timeline
- responsive layout for desktop and mobile
- accessible labels, focus states, and semantic structure

Design direction:

- operational SaaS, not a marketing landing page
- dense but readable
- polished enough to be credible as a paid product
- no generic purple gradient hero
- use multiple visual signals, not a one-note palette
